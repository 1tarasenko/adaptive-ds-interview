MVP прототипа · frontend + backend + PostgreSQL

Адаптивное техническое интервью

Интерфейс системы оценки компетенций аналитика данных: задания приходят из PostgreSQL через Python backend, ответы сохраняются, открытые ответы оценивает локальная Qwen2.5-Coder 1.5B через Ollama, BKT обновляется на сервере, а браузер собирает поведенческие признаки.

Данные кандидата

Эти данные отправляются в backend, где создаётся кандидат и сессия интервью.

Текущее задание

После отправки ответа backend проверяет тестовые задания по эталону, а развернутые ответы оценивает локальной Qwen2.5-Coder 1.5B, обновляет BKT и выбирает следующее задание.
